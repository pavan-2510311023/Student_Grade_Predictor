# 🎓 Student Grade Predictor

## 📌 Project Overview

The Student Grade Predictor is a machine learning project that predicts students' final grades based on assignment scores and attendance. It uses Linear Regression to model the relationship between academic performance indicators and final outcomes.

---

## 🎯 Objectives

* Predict student final grades
* Analyze the impact of assignment scores and attendance
* Apply basic machine learning techniques

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📊 Dataset

The dataset contains the following features:

* **Assignment Scores (0–100)**
* **Attendance (%)**
* **Final Grade (Target Variable)**

The dataset is synthetically generated for demonstration purposes.

---

## ⚙️ Methodology

1. Data generation and preprocessing
2. Feature selection (Assignment, Attendance)
3. Train-test split (80-20)
4. Model training using Linear Regression
5. Model evaluation using MAE and R2 score
6. Visualization of results

---

## 📈 Results

* Mean Absolute Error (MAE): Low error indicating good predictions
* R2 Score: High value indicating strong model performance

---

## 📉 Visualizations

* Scatter Plot: Actual vs Predicted Grades
* Histogram: Distribution of Final Grades
